#  Miniguia de Estudo: Plano Educacional Individualizado (PEI)
Projeto do Bootcamp DIO - Desafio NotebookLM


## Descrição do Projeto
Este miniguia foi criado como parte do desafio de **aprendizagem ativa com IA**. A proposta era utilizar o **NotebookLM** para curar fontes, interagir com a IA e produzir um guia de estudo estruturado sobre um tema introdutório.

O guia resultante contém:
*   **Objetivos de estudo** claros.
*   **Resumos estruturados** sobre definição, base legal, colaboração e desafios do PEI.
*   **Glossário** de conceitos-chave.
*   **Banco de prompts reutilizáveis**.

## **Conteúdo do Repositório**
Este repositório contém:
 Este arquivo README.md (instruções).
A pasta `./fontes/` com os artigos em PDF utilizados no projeto.
 Outros materiais de apoio.

## **Link do NotebookLM (Projeto Principal)**
O **caderno interativo completo** foi desenvolvido no NotebookLM e pode ser acessado publicamente através do link abaixo:
https://notebooklm.google.com/notebook/3f8ab4a9-38fb-49e8-8147-e817d094976e

> **Nota:** O caderno foi configurado com a permissão "Qualquer pessoa com o link pode visualizar". Contudo, durante os testes finais, foi identificado um comportamento inconsistente na geração do link público pela própria plataforma NotebookLM (Google), que pode solicitar login mesmo com a configuração correta. Recomenda-se que, caso o link solicite login, o acesso seja tentado a partir de uma conta Google logada para garantir a visualização completa do projeto entregue. O trabalho em si está completo e todas as evidências de configuração pública foram registradas.

## Metodologia
O projeto seguiu as etapas propostas no desafio:
1.  **Curadoria de Fontes:** Seleção de 5 fontes abertas sobre PEI e Educação Especial.
2.  **Interação com IA no NotebookLM:** Elaboração de **prompts estratégicos**, geração e registro de respostas com citações.
3.  **Organização do Conhecimento:** Estruturação das respostas em um miniguia com resumos, glossário e banco de prompts.

##  **Considerações Finais**
Este projeto demonstrou o uso do **NotebookLM como ferramenta de aprendizagem ativa**, aliando curadoria de conteúdo, pensamento crítico e organização do conhecimento.
